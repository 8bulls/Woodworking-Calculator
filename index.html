WorkShop Pro Calculator Help

üìä Basic Calculators

üßÆ Calculator Tab

Purpose: Pure math calculations with woodworking precision

Inputs: Feet, inches, fractions (1/32" to 1") or decimal inches

Operations: Addition, subtraction, multiplication, division

Features: Memory functions (MC, MR, M+, M-), history tracking, precision conversion

Best For: Total project dimensions, adding multiple measurements, quick conversions

‚úÇÔ∏è Cut Calculator Tab

Purpose: Precise cutting with automatic kerf compensation

Modes:

  • Equal Division: Divide boards into equal pieces (halves, thirds, quarters)
  • Cut Optimization: Minimize waste across multiple boards

Inputs: Board lengths, number of cuts, kerf width (1/16" to 3/16"), cut side preference

Outputs: Exact cut marks, piece lengths, total waste

Best For: Repetitive cuts, maximizing lumber efficiency

üõ†Ô∏è Specialty Calculators

üõí Lumber Shopping Cart

Purpose: Calculate lumber costs and board feet by species

Inputs: Wood species, dimensions (thickness √ó width √ó length), quantity

Outputs: Board feet, individual & total costs, shopping cart summary

Features: Species pricing memory, cart management, cost comparisons

‚ú® Golden Ratio Calculator

Purpose: Create proportionally pleasing dimensions using œÜ (1.618)

Inputs: One dimension (width or height)

Outputs: Complementary dimension, visual rectangle preview

Best For: Picture frames, furniture proportions, aesthetic design

üìê Shelf & Division Calculator

Purpose: Calculate equal spacing and divisions

Inputs: Total length, number of divisions or desired spacing

Outputs: Exact spacing, cut marks, visual diagram

Best For: Shelf spacing, picket fences, drawer dividers

üìê Miter Angle Calculator

Purpose: Calculate precise angles for joints

Modes:

  • Polygon: Regular polygons (3-12 sides)
  • Frame Builder: Custom rectangles with different angles
  • Single Cut: Individual miter cuts

Outputs: Miter saw angles, bevel angles, visual diagrams

üî© Screw & Pilot Hole Calculator

Purpose: Find correct hole sizes for any screw

Inputs: Screw size (#2-#20, fractions, lag bolts), wood type

Outputs: Pilot hole, clearance hole, countersink sizes

Features: Complete reference table, wood-specific recommendations

üìê Grade/Slope Calculator

Purpose: Calculate slopes and grades for construction

Inputs: Rise, run, or angle

Outputs: Percentage grade, angle, rise/run ratios

Features: ADA compliance indicators, common slope references

Best For: Ramps, drainage, roofing, stairs

üî≤ Tile & Paver Calculator

Purpose: Calculate tile quantities, costs, and layouts

Inputs: Tile size, project area, grout spacing, offset, waste factor

Outputs: Tiles needed, boxes required, total cost, grout estimate, cut estimation

Features: Multiple pricing methods, grout type auto-detection, perimeter validation

Best For: Flooring projects, backsplashes, outdoor patios

üå≥ Wood Species Database

Purpose: Research wood properties and characteristics

Features: 40+ species, search/filter by use case, cost level, workability

Information: Janka hardness, density, workability rating, best uses

Tools: Project wizard for wood recommendations

üìÅ Project Management System

üéØ Active Project

Select an active project from the dropdown at the top of the page. This project will:

  • Automatically receive all saved calculations
  • Show in the global selector across all tabs
  • Be used for project-specific notes and tracking

üíæ Saving Calculations

Every calculator has a "Save to Project" button that:

  • Captures complete calculation details (inputs, outputs, formulas)
  • Timestamps each save automatically
  • Allows adding custom notes
  • Shows preview before saving

What gets saved: Tool used, inputs, results, calculation method, timestamp

üìä Project Timeline

The Projects tab shows a chronological timeline of all saved calculations:

  • Filter by tool: See only specific calculator results
  • Edit/Delete: Modify saved calculations
  • Search: Find specific calculations quickly
  • Export: Download project data as JSON

Switching Projects: Use the dropdown to change active project

Project Notes: Add notes to the active project for additional context

Viewing History: Use the Projects tab dropdown to view any project's timeline

üì§ Import/Export

Import: Upload previously exported project files

Backup: Regular exports serve as project backups

üéØ Precision Settings

Set precision (1/32" to 1") to match your measuring tools. All calculations will round to your selected precision, making shop measurements easier.

üé® Themes

Choose from tool brand themes (Festool, DeWalt, Milwaukee, etc.) or general themes (Wood, Steel, Dark). Themes persist between sessions.

üì± Mobile Friendly

Designed for use in the shop with work gloves. Large buttons, clear displays, and touch-friendly interface work great on tablets and phones.

üîÑ Automatic Conversions

üíæ Data Persistence

All data (projects, calculations, preferences) saves to your browser's local storage. Works completely offline after first load.
